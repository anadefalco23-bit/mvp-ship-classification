# mvp-ship-classification

# Classificação Automática de Embarcações com Deep Learning

Este repositório apresenta um MVP desenvolvido em Python para classificação automática de embarcações utilizando técnicas modernas de Deep Learning.

## Arquiteturas avaliadas

- CNN Baseline
- MobileNetV2
- ResNet50
- YOLO11n-cls

## Tecnologias utilizadas

- Python
- TensorFlow / Keras
- Ultralytics YOLO11
- Scikit-Learn
- Pandas
- NumPy
- Matplotlib

## Resultados

O melhor desempenho foi obtido pelo modelo YOLO11n-cls, alcançando aproximadamente:

- 95,1% de acurácia na validação
- 92,96% de acurácia no conjunto de teste
- F1-score Macro ≈ 0,938
